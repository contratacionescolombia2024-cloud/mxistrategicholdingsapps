2. Click **"Copy Address"** button

### Step 3: Send USDT from Your Wallet

#### Option A: From OKX Wallet

1. Open OKX app
2. Go to **Assets** â†’ **Transfer** â†’ **Withdraw**
3. Select **USDT**
4. Choose **TRC20** network (lowest fees)
5. Paste the wallet address you copied

### Step 4: Upload Payment Proof

1. After sending, take a screenshot of:
   - Transaction confirmation screen
   - Payment QR code (if available)
   - Transaction details showing TxID

2. Back in the MXI app payment window:
   - Click **"Upload QR Code"**
   - Select your screenshot
   - Wait for upload to complete

### Step 5: Enter Transaction ID

1. In your wallet, find the transaction you just sent
2. Look for **Transaction ID** (also called TxID, Hash, or Transaction Hash)
3. Copy the transaction ID
4. Back in the MXI app:
   - Paste the transaction ID in the field
   - Click **"Verify Payment"**

### Step 6: Wait for Confirmation

**Automatic Verification (Usually 1-5 minutes):**
- System checks the blockchain
- Verifies your transaction
- Confirms payment automatically
- Updates your MXI balance
- You'll see a success message

**Manual Verification (If automatic fails):**
- Payment sent to admin for review
- Admin verifies on OKX
- Usually approved within 1-24 hours
- You'll receive notification when approved

## ðŸ†˜ Troubleshooting

### "Payment not found"

**Solution:**
1. Wait 5 minutes for blockchain confirmation
2. Check transaction status in your wallet
3. Verify you sent to correct address
4. Try verifying again
5. Contact support if issue persists

### "Transaction not confirmed"

**Solution:**
1. Check your wallet for transaction status
2. Wait for blockchain confirmation (2-10 minutes)
3. Verify transaction was successful
4. Try verifying again after confirmation

### "Amount mismatch"

**Solution:**
1. Check if your wallet deducted fees
2. System allows 1% tolerance
3. If fees > 1%, contact support
4. Provide transaction ID for manual review

### "Payment expired"

**Solution:**
1. Create a new payment
2. If you already sent funds, contact support with:
   - Transaction ID
   - Amount sent
   - Your user ID

### "Wrong wallet address"

**Solution:**
1. **DO NOT SEND** if you haven't already
2. Double-check the address in the app
3. Copy the address again
4. If already sent to wrong address, contact support immediately
